= Supprimer un certificat
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#SUPPRIMER# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/certificates/{certificate_id}`#

Supprime la ressource API de certificat spécifiée.



== Paramètres

[cols="5*"]
|===
| Nom | Type | Dans | Obligatoire | Description 


| identifiant_compte | chaîne | chemin | Vrai  a| 
ID de la ressource de compte conteneur

* exemple: {{.Account}}




| identifiant_certificat | chaîne | chemin | Vrai  a| 
ID de la collection de certificats à lister

|===


== Réponse

[source]
----
Status: 204, Indicates if the certificate resource was deleted.
----
