= Points de terminaison du backend de stockage de l'API Astra (ACS)
:allow-uri-read: 


L'API Storage Backend permet d'accéder aux représentations des fournisseurs de services de stockage (storage backends) qui peuvent être utilisées par les clusters et applications gérés.
