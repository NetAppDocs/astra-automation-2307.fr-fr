= Points de terminaison d'exécution de l'API Astra (ACS)
:allow-uri-read: 


L'API Execution Hook permet une gestion simplifiée des hooks d'exécution. Les hooks d'exécution contiennent une configuration supplémentaire et des références à une ressource source de hook.
