= Points de terminaison de liaison de rôle de l'API Astra (ACS)
:allow-uri-read: 


L'API de liaison des rôles décrit les relations entre les paires utilisateur-compte ou groupe-compte, ainsi que les ensembles d'autorisations qui leur sont accordés (rôles).
