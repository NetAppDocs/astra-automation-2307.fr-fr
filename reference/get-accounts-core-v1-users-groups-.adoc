= Récupérer un groupe pour un utilisateur
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/users/{user_id}/groups/{group_id}`#

Renvoie la ressource d'API Group stockée dans le corps de la réponse JSON.



== Paramètres

[cols="5*"]
|===
| Nom | Type | Dans | Obligatoire | Description 


| identifiant_compte | chaîne | chemin | Vrai  a| 
ID de la ressource de compte conteneur

* exemple: {{.Account}}




| ID de l'utilisateur | chaîne | chemin | Vrai  a| 
ID de la ressource utilisateur contenant



| group_id | chaîne | chemin | Vrai  a| 
ID de la collection de groupes à lister

|===


== Réponse

[source]
----
Status: 200, Returns the stored Group API resource in the JSON response body.
----
[cols="4*"]
|===
| Nom | Type | Obligatoire | Description 


| type | chaîne | Vrai  a| 
Type de média de la ressource. Les valeurs définies sont :

* "application/astra-group"




| version | chaîne | Vrai  a| 
Version de la ressource. Les valeurs définies sont :

* "1,0"
* "1,1"




| id | chaîne | Vrai  a| 
Identifiant unique global de la ressource. Les valeurs définies sont :

* Conforme au schéma UUIDv4




| nom | chaîne | Vrai  a| 
Le nom de la ressource de groupe. Si elle n'est pas spécifiée lors de la création, la valeur sera définie sur

* premier nom commun (CN) Noms distinctifs relatifs (RDN) si trouvé dans l'authID,
* authID, sinon. Si aucune valeur n'est spécifiée lors de la mise à jour, la valeur sera conservée sans modification. Les valeurs définies sont :
* Longueur minimale : 1
* Longueur maximale : 2048




| authProvider | chaîne | Vrai  a| 
Le fournisseur d'authentification pour la ressource de groupe. Les valeurs définies sont :

* "ldap"




| authID | chaîne | Vrai  a| 
Nom distinctif (DN) d'un groupe LDAP. Les valeurs définies sont :

* Longueur minimale : 1
* Longueur maximale : 2048




| les métadonnées | link:#type_astra_metadata_update["type_astra_metadata_update"] | Vrai  a| 
Métadonnées spécifiques au client et au service associées à la ressource. Les valeurs définies sont :

* Conforme au schéma de métadonnées Astra. Si aucune étiquette n'est spécifiée lors de la création, un objet de métadonnées sera créé sans étiquette. Si aucune étiquette n'est spécifiée lors de la mise à jour, les étiquettes de l'objet de métadonnées, creationTimestamp et createdBy, seront conservées sans modification.


|===
.Exemple de réponse
[%collapsible]
====
[source, json, subs="+macros"]
----
{
  "type": "application/astra-group",
  "version": "1.1",
  "id": "77754011-b93f-515e-a145-5a343aecdece",
  "name": "engineering-group",
  "authProvider": "ldap",
  "authID": "CN=Engineering,CN=groups,DC=example,DC=com",
  "metadata": {
    "labels": [],
    "creationTimestamp": "2022-10-06T20:58:16.305662Z",
    "modificationTimestamp": "2022-10-06T20:58:16.305662Z",
    "createdBy": "8f84cf09-8036-51e4-b579-bd30cb07b269"
  }
}
----
====
