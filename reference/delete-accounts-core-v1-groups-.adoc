= Supprimer un groupe
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#SUPPRIMER# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/groups/{group_id}`#

Supprime la ressource API Group spécifiée.

*Effets secondaires*

* Les ressources suivantes associées au groupe sont supprimées :
* Liaisons de rôle faisant référence au groupe




== Paramètres

[cols="5*"]
|===
| Nom | Type | Dans | Obligatoire | Description 


| identifiant_compte | chaîne | chemin | Vrai  a| 
ID de la ressource de compte conteneur

* exemple: {{.Account}}




| group_id | chaîne | chemin | Vrai  a| 
ID de la collection de groupes à lister

|===

Status: 204, Indicates if the group resource was deleted.
