= Points de terminaison de fonctionnalités de l'API Astra (ACS)
:allow-uri-read: 


L'API Feature offre un accès en lecture seule aux ressources des indicateurs de fonctionnalités afin de déterminer quelles fonctionnalités sont activées ou désactivées dans le système.
