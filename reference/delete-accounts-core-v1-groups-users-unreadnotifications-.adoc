= Supprimer une notification non lue pour un utilisateur dans un groupe
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#SUPPRIMER# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/groups/{group_id}/users/{user_id}/unreadNotifications/{unreadNotification_id}`#

Supprime la ressource API de notification non lue spécifiée.



== Paramètres

[cols="5*"]
|===
| Nom | Type | Dans | Obligatoire | Description 


| identifiant_compte | chaîne | chemin | Vrai  a| 
ID de la ressource de compte conteneur

* exemple: {{.Account}}




| group_id | chaîne | chemin | Vrai  a| 
ID de la ressource de groupe conteneur



| ID de l'utilisateur | chaîne | chemin | Vrai  a| 
ID de la ressource utilisateur contenant



| id de notification non lue | chaîne | chemin | Vrai  a| 
ID de la collection unreadNotification à lister

|===

Status: 204, Indicates if the unreadNotification resource was deleted.
