= Supprimer une sauvegarde d'application pour une application
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#SUPPRIMER# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/appBackups/{appBackup_id}`#

Supprime la ressource API de sauvegarde d'application spécifiée. Les données de sauvegarde en cours seront annulées, puis supprimées.

*Effets secondaires*

* Une fois la sauvegarde terminée, les données de sauvegarde sont supprimées.




== Paramètres

[cols="5*"]
|===
| Nom | Type | Dans | Obligatoire | Description 


| identifiant_compte | chaîne | chemin | Vrai  a| 
ID de la ressource de compte conteneur

* exemple: {{.Account}}




| app_id | chaîne | chemin | Vrai  a| 
ID de la ressource d'application contenant



| appBackup_id | chaîne | chemin | Vrai  a| 
ID de la collection appBackup à lister

|===

Status: 204, Indicates if the appBackup resource was deleted.
