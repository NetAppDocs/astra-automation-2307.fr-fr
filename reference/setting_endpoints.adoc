= Points de terminaison de configuration de l'API Astra (ACS)
:allow-uri-read: 


Un paramètre de compte est un ensemble de paires clé/valeur importantes pour une fonctionnalité connue dans un compte Astra. L'API Setting fournit un accès à des informations importantes concernant un paramètre de compte.
