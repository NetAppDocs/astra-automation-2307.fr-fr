= Points de terminaison de ressources de l’API Astra (ACS) Kubernetes
:allow-uri-read: 


L'API Kubernetes API Resource fournit un accès en lecture seule aux ressources de l'API Kubernetes afin de filtrer celles prises en charge par Astra sur un cluster donné.
