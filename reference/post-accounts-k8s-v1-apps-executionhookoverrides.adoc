= Créer une nouvelle ressource de remplacement de hook d'exécution pour une application
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-post]#POST# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/executionHookOverrides`#

Indique les valeurs souhaitées pour la ressource d'API de remplacement du hook d'exécution à créer.


CAUTION: Cette opération est obsolète.



== Paramètres

[cols="5*"]
|===
| Nom | Type | Dans | Obligatoire | Description 


| identifiant_compte | chaîne | chemin | Vrai  a| 
ID de la ressource de compte conteneur

* exemple: {{.Account}}




| app_id | chaîne | chemin | Vrai  a| 
ID de la ressource d'application contenant

|===


== Corps de la requête

Indique les valeurs souhaitées pour la ressource d'API de remplacement du hook d'exécution à créer.

[cols="4*"]
|===
| Nom | Type | Obligatoire | Description 


| type | chaîne | Vrai  a| 
Type de média de la ressource. Les valeurs définies sont :

* "application/astra-executionHookOverride"




| version | chaîne | Vrai  a| 
Version de la ressource. Les valeurs définies sont :

* "1,0"




| ID de hook d'exécution | chaîne | Vrai  a| 
Chaîne JSON contenant l'ID d'un hook d'exécution. Les valeurs définies sont :

* Conforme au schéma d'identifiants Astra. Autorise uniquement l'executionHookID d'une ressource de hook d'exécution avec une valeur hookType de « netapp ».




| activé | chaîne | Vrai  a| 
Chaîne JSON indiquant si le hook d'exécution est activé pour l'application spécifiée. Les valeurs définies sont :

* « true » - Le hook d'exécution est activé et s'exécutera dans un ordre spécifié
* « false » - Le hook d'exécution est désactivé et ne peut pas s'exécuter




| les métadonnées | link:#type_astra_metadata_update["type_astra_metadata_update"] | Faux  a| 
Métadonnées spécifiques au client et au service associées à la ressource. Les valeurs définies sont :

* Conforme au schéma de métadonnées Astra. Si aucune étiquette n'est spécifiée lors de la création, un objet de métadonnées sera créé sans étiquette. Si aucune étiquette n'est spécifiée lors de la mise à jour, les étiquettes de l'objet de métadonnées, creationTimestamp et createdBy, seront conservées sans modification.


|===
.Exemple de requête
[%collapsible]
====
[source, json, subs="+macros"]
----
{
  "type": "application/astra-executionHookOverride",
  "version": "1.0",
  "executionHookID": "eb863cc6-0d9a-454d-8ef7-81b7cac91929",
  "enabled": "false"
}
----
====
